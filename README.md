# 🐍 Exercícios Python

Repositório com exercícios de Python focados no aprendizado e prática dos principais fundamentos da linguagem e lógica de programação.

## 📚 Conteúdos abordados

- Variáveis e tipos de dados
- Strings e manipulação de texto
- Entrada e saída de dados (`input` / `print`)
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Listas
- Tuplas
- Dicionários
- Funções
- Lógica de programação
- Resolução de problemas

## 📂 Estrutura do repositório

```bash
python-exercicios/
│

```

## 🎯 Objetivo

Este repositório foi criado para consolidar conhecimentos em Python por meio da prática constante, resolução de exercícios e desenvolvimento de raciocínio lógico.
